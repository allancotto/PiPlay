PiVersus

PiVersus is a game system that uses two "sticks" as controllers to play several simple games. A raspberry pi will be kitted with a set of buttons that allow players to choose between the games. A speaker will output sound files that will explain the rules of the games, signal the start of a game and announce a winner at the end. Scoring for each game is done by real-time processing of the incoming data from the sensors in the controllers.
